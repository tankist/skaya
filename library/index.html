Skaya Zend Framework Extensions PEAR Channel Server

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover tankist.github.com/skaya
Listing available packages: 
pear remote-list -c tankist
Installing a package: 
pear install tankist/package_name
Installing a specific version/stability: 
pear install tankist/package_name-1.0.0
pear install tankist/package_name-beta
Receiving updates via a feed: 
http://tankist.github.com/skaya/feed.xml
